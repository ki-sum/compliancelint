Quick Start

Step 1 — Add to your MCP config

{
  "mcpServers": {
    "compliancelint": {
      "type": "stdio",
      "command": "python",
      "args": ["/path/to/scanner/server.py"],
      "env": { "PYTHONUNBUFFERED": "1" }
    }
  }
}

Works with Claude Code, Cursor, Windsurf, and any MCP-compatible AI client.
